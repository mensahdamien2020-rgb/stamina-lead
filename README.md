# STAMINA LEAD

Prototype d'outil d'audit agricole pour diagnostiquer la circularité, proposer des unités de micro-transformation et évaluer la rentabilité logistique. Interface et rapports en français.

Installation:

```bash
npm install
npm start
```

Pour lancer l'interface web :

```bash
npm install express
npm start
```

Ouvrez ensuite :

```bash
http://localhost:3000
```

L'interface permet de saisir des ressources localement et des paramètres logistiques. Tous les messages et rapports affichés sont en français.
